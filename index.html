Stranger Danger

The final project in the IT Capstone course at AMA Computer College.

Stranger Danger is a system prototype created for Police Officers and Civilians of Barangay San Isidro, Makati. It features an android app that civilians can use when they are in trouble or danger. Along with the help of the Stranger Danger app, responders can utilize a web interface where they can get the phone number and location of the reported danger, to help them assist the civilian in danger.

  • Date: May 2019
  • Output: Local Website and Android App (non-production)
  • Skills Utilized: HTML, CSS, JavaScript, Bootstrap, Java, Android XML, Firebase Cloud Firestore
